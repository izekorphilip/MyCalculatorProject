# Overview

As a software engineer, I am continually working to improve my understanding of programming languages and
software development concepts. The purpose of this project was to strengthen my knowledge of the Kotlin
programming language in building real world application

This software is a console-based calculator application written in Kotlin. The calculator allows users to perform
basic arithmetic operations such as addition, subtraction, multiplication, and division. It also stores a history
of calculations using collections and demonstrates the use of classes, data classes, functions, loops,
conditionals, and the `when` keyword.

My purpose for writing this software was to gain hands-on experience with Kotlin syntax and structure while
building a practical application. I also became more familiar with developing applications using IntelliJ IDEA
and working with the Java Development Kit (JDK).

[Software Demo Video](http://youtube.link.goes.here)

The software was developed using the following tools:

Programming language:
